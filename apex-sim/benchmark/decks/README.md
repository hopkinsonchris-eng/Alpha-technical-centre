# Benchmark decks

Standard SPE / community decks used for validation. Large binary/data files are
fetched, not committed; this directory documents provenance and expected paths.

| Deck  | Source | File(s) |
|-------|--------|---------|
| SPE1  | SPE comparative study 1 | `SPE1.DATA` |
| SPE3  | SPE comparative study 3 | `SPE3.DATA` |
| SPE9  | SPE comparative study 9 | `SPE9.DATA` |
| SPE10 | SPE comparative study 10 (model 2) | `SPE10.DATA`, perm/poro includes |
| Egg   | Egg model ensemble | `EGG.DATA` |
| Norne | Norne open dataset | `NORNE.DATA`, includes |

Place deck files here (or symlink) before running benchmarks. See
`docs/benchmarks/protocols.md`.
