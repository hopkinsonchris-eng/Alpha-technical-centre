# Benchmarks

Protocols and results for validating apex-sim against reference simulators
(tNavigator, ECLIPSE, OPM Flow). Decks live in `benchmark/decks/`, reference
outputs in `benchmark/reference/`, and the comparison harness in
`benchmark/compare/`.

- `protocols.md` — how each benchmark is run and what is compared.
- `results.md` — current pass/fail and error metrics vs. references.
