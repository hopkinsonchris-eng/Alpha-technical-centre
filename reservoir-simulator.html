Ensemble P10 / P50 / P90

No ensemble — run one
Each realization re-runs the full 3D IMPES engine with parameters sampled around the current base case: layer kh × lognormal(σ=0.35), porosity ± 8%, Kv/Kh × lognormal(σ=0.40), aquifer strength × lognormal(σ=0.50), skin ± 2. The seed makes an ensemble reproducible — same seed and base case give the same percentiles. P10 is the optimistic high case (exceeded with 10% probability), P90 the conservative low case.
